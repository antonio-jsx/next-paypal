# Next.js PayPal Integration

This project showcases a basic integration of PayPal with a Next.js application. You can use this as a starting point for integrating PayPal functionality into your Next.js projects.

## Installation

1. Clone the repository
```bash
git clone https://github.com/antonio-jsx/next-paypal.git
```

2. Install dependencies
```bash
npm install
```

3. Create a `.env.local` file with your PayPal client ID.
```env
PAYPAL_CLIENT_ID=
PAYPAL_SECRET=
```

4. Run the development server
```bash
npm run dev
```

5. Open your browser and navigate to `http://localhost:3000`.
