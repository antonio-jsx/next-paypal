# Next PayPal

> **Nota:** Este repositorio es una implementación básica de PayPal para aceptar pagos.

## Descripción
Next PayPal es una integración simple y ligera de PayPal diseñada para desarrolladores que trabajan con Next.js. Permite aceptar pagos de manera eficiente utilizando el conocido sistema de PayPal.

## Tecnologías utilizadas
- **TypeScript**: Lenguaje principal utilizado para proporcionar un desarrollo robusto y tipado.
- **CSS**: Para estilos del frontend.
- **JavaScript**: Usado en menor medida para funciones específicas donde TypeScript no aplica.

## Instalación
```bash
# Clona el proyecto
$ git clone https://github.com/antonio-jsx/next-paypal.git

# Instala las dependencias
$ npm install
```

## Uso
1. Configura tu cuenta PayPal en el proyecto siguiendo los pasos en la documentación oficial.
2. Ejecuta el proyecto localmente con:
```bash
$ npm run dev
```
3. Accede a la aplicación local en `http://localhost:3000` y prueba el flujo de pago con PayPal.

## Requerimientos Previos
- Tener instalado **Node.js** y **npm**.
- Cuenta sandbox de PayPal para las pruebas.
